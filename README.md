# ADFG_mapping_tools
Repository for sharing basic tools for mapping in ggplot2

Project File Structure:  
- code
- data
- figures

Workflow: 
- Store shapefiles or raster images in appropriate subdirectory of 'data'
- Follow map_template.R to create an example map
